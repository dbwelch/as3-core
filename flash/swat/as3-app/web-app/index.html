SWAT (Simple Web Application Template)

Built for Flash Player 10.0 and later player versions using Actionscript 3.

This is an example of embedding the WTG Earth application within your hosting system. Embedding of the Flash movie is done using swfobject using the static publishing method. Therefore the flashvars passed into the movie need to be duplicated as there are two object tags. The outer tag is for Internet Explorer and the inner tag within conditionals is for all other browsers.

You are free of course to change the method of embedding the Flash movie (although we think this is the best standards and browser compatible approach), but you must pass in the required flashvars to configure the application.

  • lang - The ISO 639-1 language code and ISO_3166-1_alpha-2 country code that you want the application to run in. If this is not specified the default of en_GB is used. Available values are en_GB, fr_FR, es_ES and de_DE.
  • mode - The application runtime mode, can be either auto or manual.
  • configuration - An alternative path to the XML configuration file. The default value is assets/xml/configuration.xml.

Note that if an invalid language code is supplied the Flash movie will throw a runtime exception. Also if the mode value is not auto or manual a runtime exception will be thrown.
